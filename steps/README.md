## steps(num, [start|end])

steps为animation中timing-function属性的值，有两个参数，第一个参数num是必填的，第二个参数[start|end]是选填的，当第二个参数不填时，第二个参数默认是end。第一个参数num是将动画分成num个阶段；第二个参数是动画每个阶段跳跃的位置，start表示从动画每个阶段的开始跳跃，end表示从动画每个阶段的末尾开始跳跃。
